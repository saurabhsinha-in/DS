# Java Data Structures, Design Patterns & Java 8 Best Practices

Welcome to the **Java DS Practice & Design Patterns** repository! This project helps you practice data structures and algorithms (DSA) with LeetCode and GeeksForGeeks problems, master design patterns, and implement Java 8 best practices. Most modules are validated by test cases.

## ✨ Features

- **DS & Algorithms**  
  Coding solutions and utilities for classic *LeetCode* and *GeeksForGeeks* problems.

- **Design Patterns**  
  Creational, structural, and behavioral design pattern implementations in Java.

- **Java 8 Best Practices**  
  Practical guidance and patterns with Java 8 features (lambda, streams, optionals, functional interfaces, etc.)

- **Test Validation**  
  Solutions are validated using Java test cases for reliability.

## 🚀 Getting Started

### Prerequisites

- Java 8 or higher
- Maven (preferred) or Gradle for dependency management
- Java IDE (IntelliJ IDEA recommended)

### Clone the Repository

```bash
git clone https://github.com/your-username/ds-java-practice.git
cd ds-java-practice
```

## 🛠️ How to Use

- **Explore Data Structure questions:**  
  See `src/main/java/core/`, `src/main/java/ques/algo/`, and `src/main/java/ds/udemy/ll` for implemented linked lists and algorithms.

- **Review Design Patterns:**  
  Code samples in `src/main/java/designpattern/` highlight patterns like Singleton, Builder, and Decorator.

- **Practice Java 8:**  
  The `src/main/java/version/v8/` folder holds best-practice code for Java 8 features, including functional interfaces and streams.

- **Run Unit Tests:**  
  All functionality is covered by unit tests in the `test/` directory.

## 🧪 Running Tests

Run all test cases with Maven:

```bash
mvn test
```

Or with Gradle:

```bash
gradle test
```

## 💡 Contribution Guide

- **To add a new DS problem:**  
  Create a new Java file in `core/`, `ds/udemy/ll/`, or `ques/algo/`, and corresponding tests.

- **To add a design pattern:**  
  Place implementations in `designpattern/` with examples and tests.

- **For Java 8 best practices:**  
  Add examples in `version/v8/` with appropriate documentation.

Feel free to open issues or submit pull requests for suggestions or enhancements!
